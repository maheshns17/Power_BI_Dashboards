# AI Jobs Dashboard (Power BI)


## Introduction

This dashboard was created for **Job Seekers, Career Switchers, and Data Professionals** to solve a common problem:  


Using a real-world Kaggle dataset of **4,360 AI job postings**, this project provides a single, interactive interface to explore:

- Hiring demand by role  
- Salary benchmarks  
- Experience-based compensation  
- Remote vs Onsite distribution  
- Global hiring trends  

The goal is simple: **turn raw job data into clear, actionable insights.**
