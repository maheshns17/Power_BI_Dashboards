## Introduction

This dashboard was created for **Job Seekers, Career Switchers, and Data Professionals** to solve a common problem:  


Using a real-world Kaggle dataset of **4,360 AI job postings**, this project provides a single, interactive interface to explore:

- Hiring demand by role  
- Salary benchmarks  
- Experience-based compensation  
- Remote vs Onsite distribution  
- Global hiring trends  

The goal is simple: **turn raw job data into clear, actionable insights.**

## Skills Showcased

This project demonstrates strong applied Power BI and analytics capabilities:

- ⚙️ **Data Transformation (ETL) – Power Query**: Cleaned and shaped raw job data by handling missing values, correcting data types, and creating derived columns.



- 🧠 **DAX Measures**: Created calculated measures for Average Salary, Median Salary, Min & Max Salary, Remote Job %, Total Job Count, Experience-Level Salary Analysis.  

- 📈 **Core Visualizations**: Implemented Column, Bar, Line, Area, and Scatter charts to analyze trends and salary distributions.

- 🌍 **Geospatial Analysis**: Used Map visuals to analyze global job distribution and hiring hubs.

- 📌 **KPI Indicators & Tables**: Designed dynamic KPI cards and detailed job-level tables for granular analysis.

- 🎛 **Interactive Reporting**
  - Slicers for dynamic filtering (Job Title)
  - Drill-through for detailed job-level insights
  - Buttons & bookmarks for seamless navigation
